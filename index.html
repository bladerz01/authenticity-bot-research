КОНЦЕПЦИЯ ПРОДУКТА

AuthentiBot: как это работает

Многоуровневая верификация через Telegram Mini App

📸

Уровень 1: AI-фото анализ

Пользователь отправляет фото товара. AI (Vision API) сравнивает с эталонными образцами: анализ швов, строчки, логотипов, упаковки, шрифтов. Ответ за 5–15 секунд.

Accuracy ~85–92%
🔢

Уровень 2: Штрихкод / QR / DataMatrix

Сканирование штрихкода или QR через встроенный сканер Telegram Mini App. Проверка по базам: GS1, Честный Знак (РФ), Open Food Facts, UPCItemDB и другим.

Accuracy ~95–99%
🏭

Уровень 3: Запрос производителю

Если первые два уровня дали неопределённый результат — автоматический запрос в API производителя или через стандартизированную форму. Ответ за 24–72 ч. Уникальная фича без конкурентов.

Уникально на рынке
🧬

Уровень 4: Сообщество + эксперты

База ранее проверенных товаров. Верифицированные эксперты по категориям (люкс, БАДы, косметика). Краудсорсинг: пользователи голосуют за/против подлинности.

Анализ сильных и слабых сторон

💪 Сильные стороны (S)

  • Telegram: 1 млрд аудитория без установки приложения
  • Универсальность — все категории товаров
  • Многоуровневый подход (фото + код + производитель)
  • Встроенный QR-сканер в Telegram Mini Apps
  • Низкий порог входа для пользователя
  • Честный Знак API для РФ рынка

😓 Слабые стороны (W)

  • AI-фото требует накопления обучающей базы
  • Сложно получить партнёрства с производителями
  • Accuracy фото-анализа ~85% на старте
  • Зависимость от Telegram-платформы
  • Высокие операционные расходы на экспертов

🚀 Возможности (O)

  • Рынок контрафакта РФ — $43 млрд, решения нет
  • Честный Знак расширяет список категорий ежегодно
  • Wildberries/Ozon заинтересованы в верификации
  • B2B: ретейлеры платят за такой инструмент
  • Telegram TON для микроплатежей за проверки
  • GS1 open database — 5+ млрд штрихкодов

⚠️ Угрозы (T)

  • Честный Знак может расширить функционал
  • Производители могут игнорировать запросы
  • Изменение политики Telegram для ботов
  • Сложность юридической ответственности за ошибки
  • Крупные игроки (Яндекс, Сбер) могут скопировать

ИТОГОВЫЙ ВЕРДИКТ

Целесообразность запуска

8/10

Идея имеет высокий потенциал. Рынок контрафакта в России — $43 млрд, прямых конкурентов в Telegram нет, Честный Знак покрывает лишь маркированные товары. Ключевые конкурентные преимущества: универсальность категорий + Telegram-нативность + функция запроса производителю. Главные риски: точность AI на старте и сложность партнёрств с брендами. Рекомендуется запуск в нише БАДы + косметика + одежда как первый вертикальный MVP с расширением.

✅

Почему ДА

Огромная незакрытая боль. Telegram-аудитория 1 млрд. Нет прямых конкурентов в экосистеме. Российский рынок особенно острый. Честный Знак как ready-made база.

⚡

Критические факторы успеха

Точность AI >90% с первого дня. Партнёрство с 3–5 известными брендами. Вирусный механик (поделиться результатом). B2B как якорная монетизация.

🎯

Первый шаг

Запустить MVP за 3 месяца и ~600K ₽ только со штрихкодом + Честный Знак. Проверить retention. Добавлять AI-фото итерационно по мере накопления обучающих данных.
